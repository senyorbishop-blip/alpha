# Player Guide (Founder Beta)

## Join a game

1. Open host URL from DM
2. Choose **Player**
3. Enter:
   - display name
   - Session ID
   - Player invite code

## Character/profile basics

Depending on enabled features and session settings, players can:

- maintain profile/character fields
- manage inventory/equipment
- use chat, dice, and player-side action controls

## Imports and character data

If DM has import flows enabled, follow their instructions for supported character data import.
When unavailable, manual profile + inventory entry still works.

## During play

- Move only tokens you control
- Use chat and dice tools
- Follow combat turn prompts from DM

## Reconnect flow

- If disconnected, reopen the same URL/session
- Use same account/name where possible
- Ask DM to confirm you rejoined

## Phone/tablet notes

- **Phone:** fine for joining, chat, dice, and lightweight sheet checks
- **Tablet:** better for longer sessions and inventory interactions
- **Desktop/laptop:** best overall for heavy interaction

## Founder-beta expectations

- Some features may be rough or evolving
- If anything seems stuck, refresh and rejoin first
- Report reproducible issues to your host/DM with steps
