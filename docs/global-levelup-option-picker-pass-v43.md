# Global Level-Up Option Picker Pass (v43)

This pass upgrades level-up choice cards into compare-style option cards.

## What changed
- Added richer option-card rendering for feature choices in the level-up modal.
- Added per-option guidance blocks:
  - Fantasy
  - Best fit now
  - Unlocks later
- Added a compare intro above option groups so players understand they should compare choices before locking one in.
- Added better subtype labels for:
  - Subclass options
  - Fighting styles
  - Maneuver options
  - Invocation options
  - Metamagic options
- Added targeted profile overrides for current class and custom-class option names so common picks read more like real build decisions.

## Why
The previous pass explained choice rules well, but the actual option cards still felt too flat. This pass makes subclass and class-option picks easier to compare before selection.
