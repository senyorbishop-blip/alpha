# Known Beta Limitations (Compact)

This is a quick-share summary for tester onboarding. For full details see `docs/known-issues-founder-beta.md`.

- Founder beta is stable-enough for testing, not a 1.0 polish/completeness milestone.
- Desktop is preferred for DM-heavy prep/editing workflows.
- Some optional audio/AI-enhanced features depend on external provider keys.
- Public-hosting quality depends on operator networking/proxy/TLS setup.
