# How to Report a Bug (Founder Beta)

Please include these items so issues can be fixed quickly:

1. **What you were doing** (role + goal)
2. **Exact steps** to reproduce
3. **Expected result** vs **actual result**
4. **Environment** (device, browser, desktop/phone, localhost/LAN/public)
5. **Impact level** (blocks play / workaround exists / cosmetic)
6. **Evidence** (screenshot/video/log snippet)

Use: `docs/founder-beta/bug-report-template.md`
