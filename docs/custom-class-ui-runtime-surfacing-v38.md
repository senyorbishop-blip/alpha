# Custom Class UI / Runtime Surfacing Pass v38

This pass makes the new premium custom classes feel more native in player-facing surfaces.

## Included
- Features & Traits now shows a custom class guide block for Tinker and Pirate.
- Actions now shows a custom combat-surface callout for Tinker and Pirate.
- Actions now derives resource rows from feature metadata when native resource rows are missing, which helps surface Gadget Charges and Swagger Dice.
- Level Up now shows a class-specific focus block for Tinker and Pirate.
- Removed duplicate Tracked Resources rendering from the Actions tab.

## Intent
This is a surfacing pass, not the later dedicated custom-class action pass. It improves readability and discoverability in the current app without changing the overall progression architecture.
