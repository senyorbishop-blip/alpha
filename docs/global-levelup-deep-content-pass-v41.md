# Global Level-Up Deep Content Pass v41

This pass deepens the player-facing level-up flow across all classes.

## What changed
- Added per-class choice coaching for every supported class.
- Added caster-type-aware spell-pick guidance for prepared, spellbook, and known-spell classes.
- Expanded feature cards with a `What this means for your character` block so players can understand action cost, resources, and practical impact before confirming.
- Strengthened the choices step so it reads more like a guided planner than a raw picker.

## Goal
Give players enough context inside the level-up modal that they can make informed picks without needing to jump out to other tabs.
