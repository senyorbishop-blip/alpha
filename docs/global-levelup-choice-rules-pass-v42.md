# Global Level-Up Choice Rules Pass v42

This pass deepens the level-up modal for all classes by adding rule-aware guidance around the kinds of choices players are making.

## Added
- Choice Rules section in the level-up Choices step
- Rule panels that appear when the current level includes:
  - subclass-defining choices
  - Fighting Style choices
  - maneuver choices
  - invocation choices
  - metamagic choices
  - spell swaps
  - spell picks
- Feature-specific lead text so cards explain what kind of long-term choice they represent

## Goal
Make level-up choices read more like guided build decisions instead of generic option cards.

## Effect
Players can better understand:
- whether a choice is a permanent subclass identity pick
- whether a choice is a combat posture pick
- whether a choice is a tactical resource pick
- whether a spell swap is actually worth making
