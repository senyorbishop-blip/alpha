# Custom class action pass v39

This pass makes Tinker and Pirate feel more playable in the Actions tab by surfacing authored class-specific action cards instead of relying only on generic feature fallbacks.

## Included
- Tinker class action cards for key charge / rig features
- Pirate class action cards for swagger / dirty-trick pressure tools
- Subclass action cards for the current Tinker and Pirate subclass roster
- Better action-button labels for Gadget Charges and Swagger Dice cards
- Friendlier toast fallback when a runtime action handler is unavailable

## Goal
Make the custom classes read like real combat kits in the app surfaces, not just deep data entries in the catalog.
