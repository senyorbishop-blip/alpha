# Global Level-Up Clarity Pass (v40)

This pass upgrades the **Level Up** modal for the whole class roster instead of only custom classes.

## What changed
- Added a class-specific **Level-Up Focus** guide for every supported class, including Tinker and Pirate.
- Added a clearer top-of-modal summary so players can immediately see:
  - current class level
  - next level
  - HP gain
  - how many tracked choice areas are involved
- Improved feature cards in the choice step so they now surface:
  - stronger feature headline
  - short summary when present
  - action/recovery/source tags when available
  - a clearer “choose one option” treatment for feature picks
- Reworked the review step into three readable blocks:
  - **Automatic gains**
  - **Choices still tracked**
  - **Spell plan**

## Why this matters
The main problem was that players could see a level-up modal, but it still behaved too much like a technical preview. This pass shifts the modal toward a real guided planner for all classes so players can understand what changes at the new level before they apply it.
