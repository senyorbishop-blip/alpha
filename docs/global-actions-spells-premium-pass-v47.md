# Global actions + spells premium card pass (v47)

This pass pushes the sheet closer to the polished digital-D&D feel by improving the main action and spell cards.

## What changed
- Action cards now surface four readable lanes:
  - To hit
  - Effect
  - Range
  - Spend
- Action cards now show clearer mini-pills such as Attack roll, Save effect, Bonus action, Reaction, or Usable now.
- Spell cards now surface four readable lanes:
  - Attack / Save
  - Effect
  - Range
  - Use now
- Spell buttons now read more cleanly: Cast, Attack, Roll Effect, Info.
- Premium styling tightened the hierarchy of action/spell cards so they read more like real play surfaces than generic rows.
- Runtime quick spell spotlight buttons were normalized from "Cast / Roll" to "Cast".

## Intent
This is the last broad sheet-side polish pass before moving to another major area. It focuses on:
- easier combat scanning
- better at-a-glance spell use
- clearer separation between attack, effect, range, and resource lanes
- stronger premium digital-sheet feel
