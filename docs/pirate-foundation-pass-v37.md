# Pirate foundation pass v37

This pass adds Pirate as a premium custom martial class built around swagger, mobility, dirty tricks, and battlefield presence.

## Added
- New class: Pirate
- New subclasses: Corsair, Privateer, Smuggler, Dread Captain
- Full 1–20 progression table
- Authored feature definitions for all core Pirate progression features
- Authored subclass feature definitions for all four Pirate callings

## Class identity
- Dexterity/Charisma martial controller
- d10 hit die
- medium-armor skirmisher and leader
- powered by Swagger Dice
- excels at movement, social pressure, target focus, and momentum turns

## Notable systems foundation
- Swagger Dice resource spine
- Dirty Fighting / Marked Quarry pressure loop
- boarding and terrain-aware fantasy
- subclass spread covering duelist, captain, infiltrator, and terror archetypes
