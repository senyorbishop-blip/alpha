Optional terrain texture overrides. Supported file names:
stone.png
dirt-road.png
grasslands.png
water.png
forest-ground.png
cave-stone.png
shallows-coast.png
hills.png
mountains.png
desert-sand.png
swamp-marsh.png
snow-ice.png
lava.png

If a file exists, world/tactical terrain tiles will use it instead of the procedural fallback for that terrain.
